# Windows icon catalogs

Historical Windows icon resources extracted for future ryOS asset work.

These catalogs are intentionally stored outside `public/icons` so they do not participate in active theme icon resolution or replace existing app icons.

## Catalogs

- `win98/` - Windows 98 icons from https://github.com/trapd00r/win95-winxp_icons
- `xp/` - Windows XP icons from https://github.com/trapd00r/win95-winxp_icons

Each catalog contains converted PNGs grouped into `applications/`, `control-panels/`, `dialog-ui-assets/`, `folders/`, `devices/`, `file-types/`, and `system-icons/`, plus `catalog.json` and `catalog.md` metadata with original source filenames.
