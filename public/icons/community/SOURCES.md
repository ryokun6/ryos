## Internet icon pack sources (curated subsets)

This directory contains curated subsets of internet-sourced icon packs used by CandyBar community entries.

### 1) Windows 98 (win98-rdtg)

- Source URL: https://www.wincustomize.com/explore/iconpackager/2157
- Downloaded archive: `Win98rdtg.zip`
- Original format: `.ico` (IconPackager package)
- Included subset: 10 converted PNGs in `win98-rdtg/`
- License/status: freeware pack listing on WinCustomize (no standalone license file bundled in archive)

### 2) Windows XP High Resolution (xp-highres-cc0)

- Source URL: https://github.com/marchmountain/-Windows-XP-High-Resolution-Icon-Pack
- Downloaded archive: `Windows.XP.High.Resolution.Icon.Pack.2.zip`
- Original format: `.png` icon collection
- Included subset: 10 curated PNGs in `xp-highres-cc0/`
- License/status: project page states CC0 1.0 (public domain dedication)

### 3) System 7 / Mac OS 8-9 style (system7-macos8-9-classic)

- Source URL: https://github.com/danmorgandesigns/classic-mac-elements
- Downloaded archive: `classic-mac-elements-main.zip`
- Original format: `.svg` vector icons
- Included subset: 10 SVG-derived PNGs in `system7-macos8-9-classic/`
- License/status: public GitHub repository; no standalone license file bundled in downloaded archive

### 4) macOS X Tiger/Leopard style folders (macosx-tiger-leopard-folders)

- Source URL: https://github.com/msikma/osx-folder-icons
- Downloaded archive: `osx-folder-icons-master.zip`
- Original format: iconset PNG + ICNS + PSD resources
- Included subset: 10 curated PNGs in `macosx-tiger-leopard-folders/`
- License/status: README states MIT for project files; references Apple copyright for base folder artwork

## Notes

- All images in this directory were curated/downloaded from internet sources and resized to 128x128 PNG.
- Conversion performed locally (ICO/SVG to PNG where needed).
