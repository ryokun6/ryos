# CandyBar

CandyBar is a dock customization app for ryOS: browse icon packs and apply them to the Mac-style dock. It follows the same window and sidebar patterns as other productivity apps.

## Features

- **Icon pack browser:** Explore available packs and preview styles before applying.
- **Dock integration:** Updates dock icons when a pack is selected (syncs with dock icon cloud settings where enabled).
- **Themes:** Adapts to System 7, macOS Aqua, Windows XP, and Windows 98.

## User guide

Open CandyBar from the launcher. Pick an icon pack from the list; the dock updates to match. Use the app menu for pack-related actions as exposed in your build.

## Technical notes

- Main UI: `src/apps/candybar/components/CandyBarAppComponent.tsx`
- Logic and pack types: `src/apps/candybar/hooks/useCandyBarLogic.ts`
- Registered in `src/config/appRegistry.tsx` as **CandyBar**
