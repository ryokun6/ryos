# Dashboard

Dashboard is a widget panel for ryOS that displays useful information at a glance. Press F4 or click the Dashboard icon to show your widgets—clock, calendar, and weather—and drag them to rearrange.

## Overview

Dashboard provides a quick-access overlay of widgets including an analog clock, mini calendar with events, and weather (with location permission). It’s designed for at-a-glance information without opening full apps. Widgets can be dragged to customize the layout.

## Features

- **Open Dashboard:** Press F4 or click the Dashboard icon to show your widgets
- **Clock Widget:** Analog clock showing the current time
- **Calendar Widget:** Mini calendar showing the current month and your events
- **Weather Widget:** Current weather for your location (requires location permission)
- **Move Widgets:** Drag widgets to rearrange them on the dashboard
- **Close Dashboard:** Click the background or press F4 / Escape to close

## User Guide

### Getting Started
Press F4 or launch Dashboard from the application launcher. Widgets load automatically.

### Key Actions
- **Toggle Dashboard:** Press F4 to open or close
- **Reposition Widgets:** Drag any widget to move it
- **View Weather:** Grant location permission when prompted to enable the weather widget

### Tips & Shortcuts
- F4 and Escape both close the dashboard
- The calendar widget syncs with Calendar app events

## Technical Details

### Window Configuration
Dashboard typically runs as an overlay or full-screen panel rather than a standard window.

### Component Architecture
- `src/apps/dashboard/components/DashboardAppComponent.tsx` — Main application component
- `src/apps/dashboard/components/DashboardMenuBar.tsx` — Menu bar

### Hooks & Utilities
- `src/apps/dashboard/hooks/useDashboardLogic.ts` — Dashboard state and widget layout

### State Management
Widget positions and layout preferences are persisted. Weather data is fetched based on geolocation when permission is granted.
