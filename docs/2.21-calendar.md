# Calendar

Calendar is a date management application for ryOS that lets you view months, create events, and organize your schedule with color coding. It provides month and day views with persistent event storage.

## Overview

Calendar helps ryOS users track appointments, deadlines, and events in a familiar monthly grid interface. Events are automatically saved and persist between sessions. The app supports color coding for visual organization and offers both month view for overview and day view for detailed scheduling.

## Features

- **Navigate Months:** Use arrow buttons to move between months, or click Today to jump back to the current date
- **Create Events:** Double-click a date or use File > New Event to add calendar events
- **Day View:** Click a date to select it, then switch to Day view to see all events for that day
- **Color Coding:** Assign colors to events for easy visual organization
- **Delete Events:** Select an event and use Edit > Delete Event to remove it
- **Auto-Save:** Events are automatically saved and persist between sessions

## User Guide

### Getting Started
Launch Calendar from the application launcher. Use the month navigation arrows to move between months, or click Today to return to the current date.

### Key Actions
- **Add Event:** Double-click any date in the month view, or use File > New Event
- **View Day Details:** Select a date and switch to Day view to see all events for that day
- **Organize with Colors:** Select an event and choose a color from the color options
- **Remove Events:** Select an event and use Edit > Delete Event

### Tips & Shortcuts
- Events auto-save as you create and edit them—no manual save needed
- Use colors to distinguish between work, personal, and other event types

## Technical Details

### Window Configuration
- Default size: 400×360px (typical calendar dimensions)

### Component Architecture
- `src/apps/calendar/components/CalendarAppComponent.tsx` — Main application component
- `src/apps/calendar/components/CalendarMenuBar.tsx` — Menu bar with File, Edit, and View options
- `src/apps/calendar/components/EventDialog.tsx` — Dialog for creating and editing events

### Hooks & Utilities
- `src/apps/calendar/hooks/useCalendarLogic.ts` — Calendar state and event management
- `src/apps/calendar/utils/parseIcal.ts` — iCal format parsing (if supported)

### State Management
Calendar stores events in IndexedDB or localStorage, persisting between sessions. State is managed via the useCalendarLogic hook.
