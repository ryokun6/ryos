# Control Panels

The Control Panels app in ryOS serves as the central hub for customizing your desktop environment and system preferences. It provides intuitive access to personalize your experience, from visual aesthetics to crucial system behaviors, ensuring ryOS adapts to your unique workflow and style.

## Overview

Control Panels is the primary application for managing and personalizing your ryOS experience. Users can fine-tune various aspects of the desktop environment, including its visual presentation, audio feedback, and underlying system configurations. This app is designed for both casual users looking to customize their workspace and advanced users needing to adjust core system settings or perform maintenance tasks like data backup and restoration. Its System Preferences-style pane grid groups related settings into dedicated panes so the same layout works across Aqua, classic Mac, and Windows-inspired themes.

## Features

The Control Panels app offers a comprehensive suite of features to personalize and manage your ryOS environment:

*   **Personalization & Appearance:**
    *   **Desktop Wallpaper:** Choose from a selection of photos, patterns, or dynamic shader effects to customize your desktop background.
    *   **Accent Color (Aqua & System 7):** Pick a selection/highlight color from named swatches, a wallpaper-sampled default, or **System** to restore the theme's classic look. The accent applies across menus, buttons, sidebar selections, and focus rings.
    *   **Shader Effects:** Apply unique visual enhancements like CRT, Galaxy, or Aurora effects to the ryOS interface for a distinct look.
    *   **Screen Savers:** Configure and enable various screen savers to engage when your system is idle, adding both aesthetics and privacy.
*   **Audio & Sound Management:**
    *   **Volume Mixer:** Adjust overall system volume and fine-tune specific audio outputs for a balanced sound experience.
    *   **Sound Toggles:** Enable or disable various sound effects, including general UI sounds, typing synthesizer effects, and distinct audio cues for the Terminal and Internet Explorer apps.
*   **AI Integration:**
    *   **AI Model Selection:** Choose the preferred AI model that powers intelligent features within other ryOS applications, such as Chats and the Terminal assistant.
*   **Account & Cloud Sync:**
    *   **Accounts:** Manage recovery email, Telegram linking, and account-related shortcuts from the Accounts pane.
    *   **Security:** Change or set a password, log out, log out of all devices, and delete the account from the Security pane.
    *   **Debug:** Admin users can enable Debug Mode from Accounts → Debug, which turns on runtime debug logging and the in-app console overlay.
    *   **Auto Sync:** Toggle cloud sync and choose which domains to sync in the .Mac pane. The Sync tab lists, in order: files, settings, calendar, contacts, books, maps, songs, videos, TV channels, and stickies. The cloud sync UI uses refreshed Aqua app icons.
    *   **Calendar Sync:** Enable or disable cloud sync for Calendar events from the .Mac pane.
    *   **Contacts Sync:** Enable or disable cloud sync for Contacts from the .Mac pane.
    *   **Telegram Link:** A dedicated dialog clarifies the flow for linking your ryOS account to Telegram.
*   **System Management & Maintenance:**
    *   **Backup & Restore:** Securely export all ryOS settings and user files for backup purposes, and easily restore them from a previously saved backup. Cloud backup and restore cover all sync domains: files, documents, settings, calendar, contacts, and stickies.
    *   **International (Language & Region):** A two-tab pane for localization. The **Language** tab selects the interface language and shows live date, time, and number format previews for the chosen locale. The **Date & Time** tab sets the system time zone (defaults to **Auto**, matching the browser) via a searchable list or an interactive Earth/world map, with a live local clock, date, city, and UTC-offset preview. The selected time zone is honored across ryOS clocks and the Calendar.
    *   **System Reset:** Access options to reset individual application preferences to their default state or perform a complete format of the virtual file system, effectively wiping user data and resetting the system to a clean slate.

## User Guide

### Getting Started
To launch the Control Panels app, locate its icon in the ryOS Start Menu or Taskbar. Once open, use Show All or the category grid to navigate between panes dedicated to specific areas of system configuration.

### Key Actions
*   **Customize Appearance:** Open Appearance, Desktop & Screen Saver, or Displays to modify themes, wallpaper, accent color, screen savers, shaders, and display-related preferences.
*   **Manage Sounds:** In the Sound pane, use the volume mixer to adjust audio levels. Toggle switches allow you to enable or disable specific sound effects like UI feedback, typing sounds, and application-specific audio.
*   **Select AI Model:** Admin users can open Accounts → Debug to choose the intelligence engine used by AI-powered features across ryOS applications.
*   **Manage Cloud Sync:** In the .Mac pane, enable Auto Sync and toggle individual domains (files, settings, calendar, contacts, books, maps, songs, videos, TV channels, stickies).
*   **Backup/Restore Settings:** Utilize the "Backup & Restore" options to export your current ryOS configuration and files, or import a previously saved backup to restore your settings.
*   **Change Language & Time Zone:** Open the "International" pane and use the **Language** tab to switch the ryOS interface language, or the **Date & Time** tab to pick a time zone (from the list or the Earth map) used across clocks and the Calendar.
*   **Perform System Actions:** Use the relevant maintenance actions in the .Mac and Sharing panes to back up, restore, or reset data, with confirmation dialogs for destructive operations.

### Tips & Shortcuts
*   The Control Panels app uses a pane grid with Finder-style back/forward navigation. Use Show All to return to the category grid.
*   Many settings, such as sound toggles, utilize simple on/off switches for quick and intuitive adjustments.
*   For complex settings like wallpaper selection, dedicated pickers (`WallpaperPicker`, `ScreenSaverPicker`) provide visual previews and a range of options for easy customization.
*   System-level actions, such as "Format File System," typically require confirmation dialogs to prevent accidental data loss and ensure user intent.

## Technical Details

### Window Configuration
- Default size: 400×415px
- Minimum size: 400×415px
- Maximum size: 560×600px

### Component Architecture
The app consists of 5 component file(s):

- `src/apps/control-panels/components/VolumeMixer.tsx`: Manages audio output levels and sound effect toggles for the ryOS environment.
- `src/apps/control-panels/components/WallpaperPicker.tsx`: Provides an interactive interface for selecting and applying desktop wallpapers.
- `src/apps/control-panels/components/ControlPanelsAppComponent.tsx`: The main application component, orchestrating various settings tabs, dialogs, and core logic.
- `src/apps/control-panels/components/ScreenSaverPicker.tsx`: Handles the selection and configuration of screen savers for the desktop.
- `src/apps/control-panels/components/ControlPanelsMenuBar.tsx`: Implements the application's menu bar, offering access to app-level actions like Help and About.

### Hooks & Utilities
No custom hooks or utilities are directly associated with the core Control Panels app component.

### State Management
The Control Panels app primarily manages its state using Zustand stores. It leverages `useAppStoreShallow`, `useAudioSettingsStoreShallow`, and `useDisplaySettingsStoreShallow` to interact with and persist global application, audio, and display settings across the ryOS environment, ensuring changes are saved and reflected system-wide.
